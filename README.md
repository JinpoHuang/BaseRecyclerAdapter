# BaseRecyclerAdapter
万用的RecyclerView适配器

添加依赖
```
//在工程build.gradle(Project:)下添加maven仓库
allprojects {
    repositories {
        google()
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}

//在app的build.gradle(Module:app)下添加依赖
dependencies {
    implementation 'com.github.JinpoHuang:BaseRecyclerAdapter:1.0.3'
}
```
